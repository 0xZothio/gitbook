# zOPAL Zocta Campaign

The zOPAL Zocta Campaign is the first Zocta Points campaign. It rewards users who participate directly as zOPAL LPs and users who refer new participants who become active LPs.

Referral joins alone do not generate Zocta Points. A referred user must deposit capital into zOPAL and become an active LP for referral points to begin accruing.

### Campaign Overview

| **Detail**           | **Season 1**                           |
| -------------------- | -------------------------------------- |
| Campaign             | zOPAL Zocta Campaign                   |
| TVL Cap              | $10M                                   |
| Season Length        | 180 days                               |
| Points Accrual       | Daily                                  |
| Point Status         | Locked Points                          |
| Transferability      | Non-transferable                       |
| Referral Requirement | Referred user must become an active LP |
| Join-only Referral   | No points awarded                      |
