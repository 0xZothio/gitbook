# How Referral Codes Work?

Referral codes are available to existing zOPAL LPs. Once a user becomes an LP, they can create a referral code from the Zocta Campaign dashboard and share it with new participants.

The referral flow works as follows:

1. Become a zOPAL LP.
2. Create a referral code from the campaign dashboard.
3. Share the referral code with new participants.
4. The referred user joins using the referral code.
5. Referral points begin accruing when the referred user becomes an active LP.

Referral points are linked to real capital participation. A referral joining the platform without becoming an active LP does not generate Zocta Points.
